# 2.0 新特性

- 编程语言： Java8+ 、 Kotlin 
- 底层框架： Spring Framwork5.0.x 必须要是Java8+
- 全新特性： Web Flux 新型的编程模型， 支持声明式的函数式编辑；

# 为什么选择Web Flux？

- 函数编程： Java8 Lambda 表达式的运用
- 响应编辑： Reactive Streams 
（Java9 float API ）
- 异步编程： Servlet3.1或 Asyc NIO 异常的NIO
